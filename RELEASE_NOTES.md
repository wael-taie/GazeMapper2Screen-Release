# Release Notes

## GazeMapper2Screen v1.0.0

Initial packaged Windows release.

### Included

- Qt desktop application for screen-only gaze mapping
- Left/right eye camera ROI selection
- World camera screen mapping workflow
- ArUco marker based auto-calibration target
- EyeRecToo pupil detection/tracking integration
- Screen gaze smoothing and overlay
- Self-extracting Windows package with Qt/OpenCV runtime DLLs

### Runtime Requirements

- Windows 10 or Windows 11, 64-bit
- Three compatible cameras and camera drivers
- PowerShell enabled for first-run extraction
